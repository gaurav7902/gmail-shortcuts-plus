# Gmail Arrow Key Navigation

A lightweight browser extension that enhances Gmail navigation with arrow keys.

## Features

- **Left Arrow**: Jump to the newer email
- **Right Arrow**: Jump to the older email
- **Toggle**: Enable or disable navigation on demand via the popup

## Installation

### Firefox

[![Install on Firefox](https://img.shields.io/badge/Firefox-Install%20Add--on-FF7139?logo=firefoxbrowser&logoColor=white)](https://addons.mozilla.org/en-US/firefox/addon/gmail-arrow-keys-navigator/) or [Click here](https://addons.mozilla.org/en-US/firefox/addon/gmail-arrow-keys-navigator/)

### Chromium (Chrome/Edge/Brave)

1. Download `gmail-helper-extension.zip`
2. Unzip the file to a local folder
3. Open Chrome and go to `chrome://extensions/`
4. Enable "Developer mode" (top right)
5. Click "Load unpacked" and select the unzipped folder
